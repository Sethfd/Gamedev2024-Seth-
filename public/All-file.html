Introduction to JavaScript Concepts

1. Objects

In JavaScript, an object is a collection of properties. Each property is defined as a key-value pair.


const person = {
    name: "Alice",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};

2. JSON (JavaScript Object Notation)

JSON is a lightweight data interchange format. It's easy for humans to read and write, and easy for machines to parse and generate.


const jsonString = '{"name": "Alice", "age": 30}';
const jsonObject = JSON.parse(jsonString);

3. Arrays

An array is a list-like object used to store multiple values in a single variable.


const fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]); // Outputs: Banana

4. CSS

CSS (Cascading Style Sheets) is used to style HTML documents. You can set colors, fonts, layouts, and more.


h1 {
    color: blue;
}

6. Events

Events are actions that occur in the browser, like clicks or keyboard input. You can use JavaScript to respond to these events.


document.getElementById("myButton").addEventListener("click", function() {
    alert("Button was clicked!");
});
